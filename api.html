Sensors Information History

Link

Perform GET request to /api/sensors/history

Use query string to specified the following parameters (all optional):

Type: Datetime (ISO-8601 standard)

  • from: only query for the sensor information after this time
  • to: only query for the sensor information before this time
    •

    Type: Integer

  • limit: Specify number of sensor information per page [Default: 50]
  • page: Specify page [Default: 0]
